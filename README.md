# ConsoleTetris
一个基于Windows控制台的俄罗斯方块游戏。<br>
它除了提供基本俄罗斯方块游戏以外，还有双人模式、隐形模式、道具系统、键位设置、界面宽度设置等功能。
这是我的早期作品之一，编写于2016年6月，完成于2016年10月。
